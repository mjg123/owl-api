Fact 2: Cats and dogs have their own APIs
Fact 3: Owls now have their own API too!

The API

Making HTTP GET or POST requests against https://owls.gilliard.lol/api/owlfacts will get you facts about owls.
You can fetch up to ten facts at once, for example https://owls.gilliard.lol/api/owlfacts?count=5
The Owl Facts API is only available over HTTPS. Requests over HTTP will be redirected.
