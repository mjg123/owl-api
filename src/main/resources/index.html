Fact 2: Cats and dogs have their own APIs

The API

Owl Facts

Making HTTP GET or POST requests against https://owls.gilliard.lol/api/owlfacts will get you facts about owls.
You can fetch up to ten facts at once, for example https://owls.gilliard.lol/api/owlfacts?count=5
The Owl Facts API is only available over HTTPS. Requests over HTTP will be redirected.
Owl facts are only available in the English language at the moment.

Owl Pics

If you just want to look at an owl, I don't blame you at all. Head to https://owls.gilliard.lol/api/owlpic.
Owl pictures are sourced from various places on the internet. All pictures have been cleared for re-use and the API returns an `X-Attribution` header with each image.
